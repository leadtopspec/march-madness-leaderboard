# 🏀 March Madness Leaderboard

A real-time competition tracking platform for sales tournaments and March Madness competitions.

![March Madness Demo](https://march-madness-leaderboard.vercel.app)

## 🎯 Features

### Real-Time Competition Platform
- **Live Leaderboard** - Updates instantly as scores are submitted
- **Tournament Bracket Visualization** - 32-team bracket with seeding
- **TV Display Mode** - Full-screen view perfect for mounting on displays
- **Mobile Score Submission** - Easy-to-use interface for recording results
- **Real-Time Notifications** - Instant updates without page refresh

### Professional Competition Management
- **Agent Dashboard** - Individual performance tracking
- **Matchup System** - Head-to-head competition display
- **Score Tracking** - Comprehensive sales/performance analytics
- **Tournament Progression** - Automatic advancement through bracket rounds
- **Admin Controls** - Manage participants and competition settings

## 🏃‍♂️ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Local Development
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/march-madness-leaderboard.git

# Navigate to project
cd march-madness-leaderboard

# Install dependencies
npm install

# Start development server
npm run dev

# Open browser to http://localhost:3000
```

### Build for Production
```bash
# Build the application
npm run build

# Start production server
npm start
```
